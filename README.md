# Signal Types Classification
Проект посвящён задаче автоматической кластеризации сигналов сцинтилляционного детектора с использованием методов машинного обучения без учителя (unsupervised learning).

# Описание проекта
В работе исследуются сигналы, полученные со сцинтилляционного детектора на основе органических кристаллов паратерфенила. Основная задача заключается в разделении сигналов по их физической природе.

Необходимо кластеризовать сигналы на три группы:
- Кластер 0 — сигналы первого типа частиц;
- Кластер 1 — сигналы второго типа частиц;
- Кластер 2 — аномальные или неидентифицируемые сигналы.

Проект выполнен в рамках соревнования Kaggle: Signal Types Classification
https://www.kaggle.com/competitions/signal-types-classification


# Просмотр проекта
Ноутбук project.ipynb

# Научная и практическая значимость
Сцинтилляционные детекторы широко используются:
- в ядерной энергетике;
- в системах радиационного мониторинга;
- в регистрации нейтронного и гамма-излучения;
- в физических экспериментах;
- в медицине и промышленности.

# Автоматическое разделение сигналов позволяет:
- повысить точность анализа излучения;
- автоматизировать обработку больших объёмов данных;
- выявлять аномальные сигналы;
- улучшать системы радиационного контроля.

# Используемые методы
- Предобработка сигналов
- коррекция базовой линии;
- нормализация сигналов;
- выделение полезной части импульса.


# Feature Engineering
Для каждого сигнала вычислялись физически интерпретируемые признаки:
- r_long описывает долю хвостовой части импульса,
- r_short - долю быстрой компоненты,
- ratio - показывает их относительное соотношение. 
- Дополнительно рассчитываются признаки **asymmetry** и **log_amp**.

# Использовались методы:
- PSD (Pulse Shape Discrimination);
- PCA (Principal Component Analysis);
- интегральные характеристики сигнала;
- экспоненциальный анализ хвоста сигнала.

# Используемые модели
В проекте были протестированы:
- KMeans;
- Gaussian Mixture Model (GMM);
- PCA-based clustering;
- Isolation Forest;
- Local Outlier Factor (LOF).


# Основные этапы работы
1. Исследовательский анализ данных (EDA)
2. Предобработка сигналов
3. Feature Engineering
4. Кластеризация, обучение и выбор лучшей модели
5. Подбор гиперпараметров и итоговое предсказание по модели GMM (все признаки_Final_ML)
6. Итоговый вывод


# Результаты
В ходе исследования было показано, что:
- физически интерпретируемые признаки значительно улучшают качество кластеризации;
- PCA и GMM позволяют хорошо разделять сигналы;
- время спада сигнала является одним из наиболее информативных признаков;
- методы поиска аномалий позволяют выделять нестандартные сигналы в отдельный кластер.
